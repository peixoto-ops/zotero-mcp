# STEPS
1. Analisar o documento jurídico fornecido

3. Extrair elementos obrigatórios: tribunal, relator, data, processo, ementa

5. Incluir campos opcionais relevantes (ementa, precedentes citados, etc.)

# OUTPUT INSTRUCTIONS
- A saída deve ser uma entrada BibTeX válida
- Usar tipo @misc para decisões judiciais
- Incluir campos: author, title, year, month, day, howpublished, note
- Não incluir comentários ou explicações adicionais
- Manter formatação consistente com padrões Zotero
