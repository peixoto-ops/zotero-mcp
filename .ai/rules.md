# Regras Gerais do Sistema Cognitivo

1. Toda saída deve ser estruturada.
2. Nenhum agente pode produzir texto livre sem schema ou formato definido.
3. Patterns devem ser reutilizáveis e documentados.
4. Fluxos devem ser explícitos e versionados.
5. Lógica cognitiva não deve ficar apenas no código.
6. Sempre que possível, validar saídas com JSON Schema.
