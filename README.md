# zotero-mcp - Sistema de Orquestração Jurídica com Model Context Protocol

Este projeto implementa um sistema completo de orquestração jurídica assistida por múltiplos agentes, integrado ao **Zotero**, **Obsidian** e **Fabric**, com foco em:

- análise profunda de precedentes jurídicos
- construção controlada de teses jurídicas
- redação modular e verificável de peças processuais
- preservação de cadeia de custódia e rastreabilidade probatória

## Escopo do Projeto

O sistema é um **orquestrador de produção jurídica** que:

- **Integra MCPs (Model Context Protocols)** para conectar diferentes ferramentas cognitivas
- **Automatiza fluxos jurídicos** com agentes especializados (precedentes, contexto, estratégia, redação, provas)
- **Utiliza o Fabric** para padrões cognitivos especializados em análise jurídica
- **Sincroniza com Zotero** para gerenciamento de referências bibliográficas
- **Conecta ao Obsidian** para armazenamento e organização de conhecimento
- **Implementa subagentes** com responsabilidades bem definidas e orquestração inteligente

## Componentes Principais

- **Agentes Especializados**: Precedentes, Contexto, Estratégia, TOC, Redação Jurídica, Prova e Cadeia de Custódia, Zotero
- **Especificação Cognitiva**: Pasta `.ai/` com contratos formais de análise e raciocínio
- **Padrões do Fabric**: Templates cognitivos para tarefas jurídicas específicas
- **Schemas de Validação**: Controle de qualidade e prevenção de alucinações
- **Fluxos Automatizados**: Pipelines completos de processamento documental

## Estrutura do Projeto

- `.ai/` - Infraestrutura cognitiva (README, identity, rules, GOVERNANCE, agentes, fluxos, patterns, schemas, prompts, AGENT_PROMPT, catalog, orchestrators, teams, consolidation, integration)

## Objetivo

Criar um sistema de engenharia jurídica com:
- separação clara de responsabilidades
- auditabilidade completa
- rastreabilidade de decisões
- redução de risco de erro ou alucinação
- integração fluida entre ferramentas de IA e conhecimento jurídico
